# Installation
In order to install requirements, follow:
```
pip install scipy==1.6.3
pip install stable-baselines3[extra]
apt-get install -y xvfb python-opengl > /dev/null 2>&1
pip install gym pyvirtualdisplay > /dev/null 2>&1
pip install -U kora
```
